# Selfly 心理测试项目需求分析

## 📋 项目概述
- **项目名称**: Selfly - 心理测试小程序
- **项目类型**: 心理测试类应用
- **目标平台**: 微信小程序
- **技术栈**: uni-app + Vue3 + TypeScript + UnoCSS + Pinia
- **商业模式**: 广告变现 + 增值服务

## 🎯 核心目标
1. **吸引流量**: 通过免费的心理测试吸引用户
2. **提高粘性**: 测试结果分享、历史记录、个性化推荐
3. **商业变现**: 广告展示、付费深度测试、专业报告
4. **社交传播**: 结果分享、好友对比、排行榜

## 👥 目标用户
- **主要用户群体**: 18-35岁年轻人，对自我探索感兴趣的群体
- **用户特征**: 喜欢分享、好奇心强、有一定消费能力
- **用户需求痛点**:
  - 了解自己的性格特征
  - 寻找职业发展方向
  - 人际关系改善需求
  - 娱乐消遣需求

## 🔧 技术需求
- **性能要求**: 启动时间 < 2s，页面切换流畅
- **兼容性**: 微信版本 7.0+
- **存储方案**: 本地存储 + PostgreSQL数据库
- **安全要求**: 数据加密传输，用户隐私保护

### 🗄️ 数据库设计
- **数据库类型**: PostgreSQL (网络云数据库)
- **ORM框架**: SqlSugar (.NET Core)
- **主要数据表**:
  - users (用户表)
  - tests (测试表)
  - questions (题目表)
  - test_results (测试结果表)
  - user_test_records (用户测试记录)

### 🌐 后端API设计
- **技术栈**: .NET Core + SqlSugar
- **API风格**: RESTful API
- **主要功能**:
  - 用户认证和信息管理
  - 测试数据CRUD操作
  - 测试结果计算和存储
  - 数据统计和分析

### 📱 数据交互方案
- **前端**: uni-app (小程序端)
- **后端**: .NET Core API
- **通信**: HTTP/HTTPS + JSON
- **缓存策略**: 本地存储 + Redis(可选)

## 🎨 设计规范
- **主色调**:
  - 主色: #6366F1 (靛蓝色，代表理性和洞察)
  - 辅色: #EC4899 (粉红色，代表情感和温暖)
  - 中性色: #F8FAFC (背景), #1E293B (文字)
- **字体规范**: 微信默认字体 + 数字使用 SF Mono
- **图标风格**: Linear 24px 统一风格
- **交互模式**: 卡片式设计 + 流畅过渡动画

## 🎯 测试实现顺序 (按优先级)
1. **PDP动物性格测试** ⭐ (首个实现)
   - 简单有趣，只有5种动物类型
   - 题目数量少 (约40题)，容易完成
   - 结果形象生动，易于分享传播
   - 适合作为测试引擎的验证项目

2. **MBTI十六型人格测试** ⭐ (第二个实现)
   - 最流行的性格测试，用户认知度高
   - 16种人格类型，内容丰富
   - 社交分享属性强，流量效果好
   - 题目量大 (60-80题)，用户粘性强

3. **暗黑人格测试** (第三阶段)
   - 话题性强，容易引发讨论
   - 有争议性，传播效果好

4. **韦氏智力测试** (后期完善)
   - 专业性强，有付费潜力
   - 可以作为增值服务

5. **九型人格测试** (后期完善)
   - 深度分析，复购率高
   - 适合心理学爱好者

## 💡 关键成功因素
1. **用户体验**: 答题过程流畅，结果展示精美
2. **社交属性**: 结果分享要美观且吸引点击
3. **算法准确性**: 测试结果要可信，有科学依据
4. **广告平衡**: 不影响用户体验的前提下合理变现
5. **内容更新**: 定期增加新测试保持新鲜感
